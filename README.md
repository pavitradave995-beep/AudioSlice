# 🎵 AudioSlice

### Cut, Combine, and Create Your Perfect Audio Moments.

AudioSlice is a professional AI-powered audio editing application built with Streamlit. It enables you to upload audio files, select favorite sections, trim unwanted parts, combine multiple clips, and export the final result.

## ✨ Features

### Core Features
- **Audio Upload**: Support for MP3 and WAV files (up to 100MB)
- **Interactive Waveform**: Visual waveform with zoom and selection
- **Precision Editing**: Millisecond-level clip creation
- **Clip Management**: Create, edit, and delete unlimited clips
- **Merge & Export**: Combine clips and export as MP3 or WAV
- **Project Management**: Save and load projects as JSON files

### AI Features
- **Smart Audio Analysis**: Detect repeated sections and chorus patterns
- **Highlight Detection**: Identify high-energy segments
- **Intelligent Recommendations**: Get AI-suggested clip boundaries with confidence scores

### User Interface
- **Dark Mode**: Professional dark theme for comfortable editing
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Clean cards, smooth layouts, and premium typography
